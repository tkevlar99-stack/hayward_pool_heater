# Success Story Assets

Use this folder for small images, wiring diagrams, sanitized logs, or other files that directly support a success story.

Before adding assets:

- Confirm the contributor intended the asset to be included in the repository.
- Preserve attribution in the story that references the asset.
- Remove private information such as Wi-Fi names, passwords, API keys, tokens, serial numbers if sensitive, GPS data, addresses, faces, and unrelated personal details.
- Prefer links to the original GitHub issue attachments for large YAML files, code dumps, or many photos.
- Use descriptive filenames such as `pc1001-cn1-wiring-ed-fr.jpg` or `pc1001-startup-log-ed-fr.txt`.

Do not add generated ESPHome build output, firmware binaries, PlatformIO caches, or local secrets.
